ESED Connect - Guia do App

O que é o ESED Connect?

É um ecossistema digital inovador projetado especialmente para a comunidade escolar da Escola Secundária Emília Daússe (ESED). Centraliza avisos, cronogramas e livros didáticos.

Como poupar dados (Megabytes)?

O aplicativo foi otimizado para a realidade local. Ativa o Modo Poupança de Dados nas definições para bloquear imagens pesadas e navegar usando apenas texto leve.

Níveis de Segurança

Cada perfil de acesso (Estudante, Chefe de Turma, Direção ou Programador) desbloqueia funções personalizadas no menu de forma imediata.
